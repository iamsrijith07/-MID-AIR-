## 🌐 Live Demo

[👉 Try Air Draw Now!](https://iamsrijith07.github.io/-MID-AIR-/)


MID AIR turns your webcam into an interactive canvas. By leveraging real-time hand-tracking technology, this browser-based app maps specific hand gestures to drawing actions. There are no heavy frameworks involved—just pure web technologies and computer vision working together to create a magical, augmented-reality doodling experience.

✨ How It Works (The Gestures):

☝️ Index Finger Up: Draw neon glowing strokes.

🖐️ Open Palm: Act as an eraser to sweep away mistakes.

🤏 Pinch (Thumb & Index): Grab and drag existing strokes around the screen.

✊ Closed Fist: Idle mode (rest your hand without drawing).

🚀 Features:

Real-time hand landmark tracking with zero latency.

Glassmorphic, dark-mode user interface.

Customizable color palette and brush thickness.

Fully responsive layout for different screen sizes.

💻 Tech Stack:

Frontend: Vanilla JavaScript, HTML5 Canvas, and CSS.

Computer Vision: Google MediaPipe Tasks Vision API (Hand Landmarker).
